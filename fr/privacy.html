Politique de confidentialité

Dernière mise à jour : Mars 2026

EAD Studio ("nous") développe l'application mobile Familoo. Cette politique explique comment nous collectons, utilisons et protégeons vos informations.

📋 Informations collectées

Compte : Lors de la création d'une famille, nous collectons votre email Google uniquement pour l'authentification.

Données familiales : Événements, noms des membres et paramètres sont stockés sur Firebase (Google Cloud).

Usage anonyme : Les membres rejoignant via QR code utilisent une authentification anonyme — aucune donnée personnelle requise.

🔧 Utilisation des informations

  • Fournir et maintenir le service Familoo
  • Synchroniser le calendrier entre les appareils
  • Envoyer des notifications (si activées)

Nous ne vendons jamais vos informations personnelles.

💾 Stockage

Vos données sont stockées sur Firebase (Google Cloud), conforme ISO 27001, SOC 1/2/3.

🌐 Transfert international

Les serveurs Firebase peuvent être hors UE. Google adhère aux clauses contractuelles types de l'UE et au Data Privacy Framework.

⏱️ Conservation

Données conservées tant que le compte est actif. Suppression de la famille = suppression des données sous 30 jours.

📢 Publicité

Version gratuite : bannières via Google AdMob. Désactivez la personnalisation dans les paramètres de votre appareil. Supporters = sans publicité.

💳 Paiements

Gérés par RevenueCat et Google Play / Apple App Store. Nous ne stockons aucune information de paiement.

⚖️ Vos droits (RGPD)

  • Accès : Obtenir une copie de vos données
  • Rectification : Corriger vos données
  • Effacement : Demander la suppression
  • Portabilité : Exporter vos données
  • Opposition : Retirer votre consentement

Contact : easyapplicationdev@gmail.com

👶 Mineurs

Familoo est conçu pour un usage familial. Les enfants rejoignent via QR code avec authentification anonyme.
